# LinearGradient

![screenshot](https://i.imgur.com/LYPglYDm.jpg)

Utility extensions for Unity standard Gradient class

## How to install

You can install the LinearGradient package (`jp.keijiro.klak.lineargradient`)
via the "Keijiro" scoped registry using the Unity Package Manager. To add the
registry to your project, follow [these instructions].

[these instructions]:
  https://gist.github.com/keijiro/f8c7e8ff29bfe63d86b888901b82644c
